## 介绍

我发现我不喜欢算法题一个原因是确实太难了，懒懒散散的；
第二个问题就是刷题比较繁琐，在leetcode官方上做题，遇到代码跑不过测试用例的时候，需要在编辑器里面创建java文件，然后把代码拷过来，在自己编写测试用例等等，这些让我一而再再而三的放弃算法的学习；在加上现在大模型的崛起，更加削减了对算法的热情。
现在构建出来自己的刷题流程，持之以恒。
